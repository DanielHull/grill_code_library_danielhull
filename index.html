The purpose of this matlab project is to ...

1) provide an edit link

2) add on a link for a function prototype and retrieve that line where the function is defined

3) add an indicator if the method is static

4) utilize the profiler to help a developer make code faster

This code continues to develop. It presents minor issues now specifically related to the period link. Continued monitoring and reworking updated daily.

Insertion of the metaclass functionality to create a period link for referencing the function prototype of inputs, outputs, and file pathway.

Seperation string generated with the edit colon link has been completed.
